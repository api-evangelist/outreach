## APIs
- **Outreach REST API** — JSON:API-flavored REST endpoints covering accounts, prospects, sequences, mailings, calls, opportunities and tasks.
- **Outreach Client Extensions API** — Embed UI extensions and add custom actions in the Outreach app.
- **Outreach Data Sharing API** — Bulk data sharing and integration capabilities.
